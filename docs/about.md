# About This Textbook

## Educational Methodology

This textbook applies **Bloom's Taxonomy (2001 Revision)** as its pedagogical framework. Each chapter targets a specific cognitive level:

| Chapter | Bloom's Level | Cognitive Goal |
|---------|--------------|----------------|
| 1. Real Estate Investment Fundamentals | Remember | Define, list, identify |
| 2. Market Analysis and Property Evaluation | Understand | Explain, distinguish, interpret |
| 3. Financial Metrics — Cap Rate, NOI, Cash-on-Cash | Apply | Apply, analyze, compare |
| 4. Financing Strategies and Leverage | Analyze | Analyze, evaluate, distinguish |
| 5. The BRRRR Method | Evaluate | Evaluate evidence, judge significance |
| 6. Property Management and Operations | Create | Design frameworks, integrate strategies |
| 7. Tax Strategy and Portfolio Scaling | Evaluate/Create | Create frameworks, critically assess |

## Intelligent Textbook Approach

This is a **Level 3 intelligent textbook** featuring:

- **Interactive MicroSims** — Browser-based simulations for hands-on exploration
- **Structured navigation** — Concept dependencies respected across chapters
- **Rich formatting** — Admonitions, diagrams, and comparison tables
- **Comprehensive glossary** — ISO 11179-compliant definitions
